# FORGE+C 2.0 Workflow — Chi Tiết

## 8 Bước Sản Xuất (V1.2A)

```
F(Frame) → O(Order) → R(×3 tuần tự + MIX) → G(Grind) → E(Elevate) → C(Cross-check) → +C(Council)
```

### F — Frame (1 agent)
- Skeleton key cho chương
- Chia 5 beats
- Micro-outline 50-100 từ/beat
- Câu mở + câu đóng
- Motif/callback cần

### O — Order (1 agent)
- Tối ưu thứ tự beats
- Phi tuyến? Flashback? Mở giữa?
- Tension curve
- Output: BEAT BRIEFS

### R — Render (3 agents TUẦN TỰ × 5 beats + MIX)
- Agent 1: viết từ brief (độc lập)
- Agent 2: đọc bản 1 tham khảo → viết TỰ DO
- Agent 3: đọc bản 1+2 tham khảo → viết TỰ DO
- Micro-council EDITOR: chọn khung + MIX highlights

### G — Grind (1 agent)
- Ghép 5 MIX beats
- Check: 0 EN, 0 markdown, 0 sến
- Check: 0 lặp, continuity, giọng

### E — Elevate (1 agent)
- Transitions giữa beats
- Callbacks
- Pacing
- Thêm hài đen nếu thiếu

### C — Cross-check (1 agent)
- So outline
- Novel Bible nhất quán
- Chương trước continuity
- Running gags

### +C — Council (1 agent, 5 vai)
- Nacharium thẩm định
- PASS ≥ 7/10
- AI traces check
