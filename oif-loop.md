# OIF Infinity Loop — Chi Tiết 6 Phases

```
SEED → NOVEL BIBLE → FORGE+SEQ → GHÉP+EDITOR → NACHARIUM → SELF-ABSORB → LOOP
```

## Phase 1: SEEDING (5 phút)
- Seed ý tưởng
- Check trùng (web search)
- Đủ mới → auto Phase 2
- Trùng → seed lại tự động

## Phase 2: NOVEL BIBLE (5-10 phút)
7 mục:
1. Character Sheets
2. World Rules
3. Resonance Map
4. Skeleton Key
5. Tone Compass
6. Emotional Map
7. Series Arc

## Phase 3: FORGE+SEQ (45-60 phút)
- 5 beats × 3 agents TUẦN TỰ
- Micro-council EDITOR MIX mỗi beat
- = 15 bản render + 5 bản MIX

## Phase 4: GHÉP+EDITOR (15-25 phút)
3 bước riêng:
1. Grind (sạch lỗi)
2. Elevate (nâng tầm sáng tạo)
3. Cross-check (nhất quán tổng thể)

## Phase 5: NACHARIUM (10-15 phút)
- Council 5 vai thẩm định
- AI traces check
- PASS ≥ 7/10

## Phase 6: SELF-ABSORPTION (5 phút)
- Hấp thụ bài học
- Ghi memory
- Báo cáo
- Seed chương tiếp → Loop
